# desk_pet
Bocchi-chan Desktop Pet  A desktop pet based on **Hitori Gotoh (Bocchi)** from the anime **Bocchi the Rock!**.  I made this as an **18th birthday gift** for a childlike friend who loves the show.
## Note on Assets

The original character artwork is not included in this repository out of respect for the artist and as a personal gift for my friend. 

**See the demo video below** to see how the pet looks and behaves.
